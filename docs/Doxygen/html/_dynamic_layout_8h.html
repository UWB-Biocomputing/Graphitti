Detailed Description

The DynamicLayout class defines the layout of neurons in neural networks.

/

The DynamicLayout class maintains neurons locations (x, y coordinates), distance of every couple neurons, neurons type map (distribution of excitatory and inhibitory neurons), and starter neurons map (distribution of endogenously active neurons).

The DynamicLayout class generates layout information dynamically.
