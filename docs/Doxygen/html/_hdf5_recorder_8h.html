Detailed Description

An implementation for recording spikes history on hdf5 file.

/

The Hdf5Recorder provides a mechanism for recording neuron's layout, spikes history, and compile history information on hdf5 file:

  1. neuron's locations, and type map,
  2. individual neuron's spike rate in epochs,
  3. network wide burstiness index data in 1s bins,
  4. network wide spike count in 10ms bins.

Hierarchical Data Format (HDF) is a set of file formats (HDF4, HDF5) designed to store and organize large amounts of data.
