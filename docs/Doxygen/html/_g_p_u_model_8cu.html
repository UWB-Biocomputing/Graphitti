Calculate the sum of synaptic input to each neuron.

Calculate the sum of synaptic input to each neuron. One thread corresponds to one neuron. Iterates sequentially through the forward synapse index map (synapseIndexMapDevice_) to access only existing synapses. Using this structure eliminates the need to skip synapses that have undergone lazy deletion from the main (allEdgesDevice) synapse structure. The forward map is re-computed during each network restructure (once per epoch) to ensure that all synapse pointers for a neuron are stored contiguously.
