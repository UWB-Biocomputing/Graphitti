◆ spikeHistory_

uint64_t** AllSpikingNeurons::spikeHistory_

Step count (history) for each spike fired by each neuron. The step counts are stored in a buffer for each neuron, and the pointers to the buffer are stored in a list pointed by spike_history. Each buffer is a circular, and offset of top location of the buffer i is specified by spikeCountOffset[i].
