Detailed Description

Implementation of Model for the graph-based networks.

/

The Model class maintains and manages classes of objects that make up essential components of the graph network.

  1. AllVertices: A class to define a list of particular type of vertices.
  2. AllEdges: A class to define a list of particular type of edges.
  3. Connections: A class to define connections of the graph network.
  4. Layout: A class to define vertices' layout information in the network.

Edges in the edge map are located at the coordinates of the vertex from which they receive output.

The model runs on multi-threaded on a GPU.
